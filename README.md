# Project Pizzeria

Projekt zaliczeniowy modułu Fullstack JS w ramach Kodilla Fullstack JS Bootcamp — aplikacja pizzerii z menu, koszykiem, systemem rezerwacji stolików oraz stroną główną.

## Live demo

Strona jest opublikowana na Replit i dostępna pod adresem:

https://replit.com/@cimmerian422/project-pizzeria

## Funkcjonalności

- Menu produktów z konfigurowalnymi opcjami i koszykiem
- System rezerwacji stolików (wybór daty, godziny, liczby osób, dodatków) z podglądem dostępności w czasie rzeczywistym
- Strona główna z karuzelą opinii, sekcją CTA oraz galerią zdjęć

## Uruchomienie lokalne

```bash
npm install
npm run watch
```
